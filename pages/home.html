My Project

Blanja Web

Blanja is an E-Commerce website that is used to make it easier for users to shop without leaving the house, it can be used anytime and anywhere. Here there is a register and login feature for sellers and customers, As a seller you can add, update, and delete products being sold, and as a customer, you can buy goods provided by the seller.

This website was created using React JS for the frontend and Express JS for the backend, already deployed with Vercel.
